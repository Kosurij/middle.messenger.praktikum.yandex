## Локальный запуск.

1. `npm install` — Установка необходимых зависимостей.
2. `npm start` — запуск проекта с отдачей статики локальным сервером (localhost:3000).
3. `npm dev` — запуск проекта для разработки сборщиком parcel (localhost:1234).
